Quick background

I'm a modern languages graduate with a bunch of (nerdy) interests, and whilst I've worked in marketing for a number of years, now I'm taking the exciting leap into Web development!

Teenage forays into HTML during the <table> heydays of the Naughties and modding the videogame Warcraft III piqued my early interest and understanding of all things Code-y; but an uninspiring GCSE ICT course making spreadsheets for clothes stores led me to prefer a more human type of 'code' - languages.

A degree in the wrong type of language

One First Class Honours Degree from Durham University in Spanish and Italian later and I'm back making spreadsheets for a clothes store, albeit with a sprinkling of VBA. But it's Web Development I want to be doing. So I set about doing it.

A future in development

Aside from making swish spreadsheets to manage stock, Amazon listings, and SEO content for my brother's fashion brand University of Whatever, I also built the website. A baptism by fire, perhaps, but doing everything from installing WordPress on the server to styling the About Us page gave me a broad experience in managing an eCommerce site based on a CMS as well as using web languages HTML and CSS.

Meanwhile I've been studying an online course created by Rob Percival to get a more in-depth knowledge of key languages, concepts, and workflow processes within web development - including HTML5, CSS3, JavaScript, jQuery, Bootstrap, PHP, and MySQL.

To demonstrate and solidify my learning so far, I built this site from scratch and published it through GitHub Pages. I documented the process I took - from wireframes in PhotoShop to polishing the style in CSS - and have posted it on this site as a reference for myself and as evidence that I do have some idea what I'm talking about. I urge you to check it out, and you can find all the code on my GitHub profile.
